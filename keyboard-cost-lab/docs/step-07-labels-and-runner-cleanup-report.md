# 7단계 보고서: 버튼 이름 구분 가능성 보완과 수행 탭 정리 리팩토링

## 1. 이번 단계 목표

이번 단계의 목표는 두 가지였다.

1. 홈 화면의 서비스 카드 버튼 이름이 서로 구분되도록 바꾼다.
2. 수행 탭 안에 남아 있던 중복 이동 제어와 실험 설명 문구를 정리하고, 공통 코드를 묶어 유지 보수 부담을 줄인다.

사용자 요청의 핵심은 탭 키로 이동할 때 버튼 이름만 읽어도 대상을 구분할 수 있어야 하며, 수행 탭에서는 실제 서비스 콘텐츠와 맨 아래 보조 영역만 남아 있어야 한다는 점이다.

## 2. 문제 확인 결과

이전 단계 기준 소스 코드를 기준으로 다음 문제를 확인했다.

### 2.1 홈 화면 버튼

`app.js`의 서비스 카드 렌더링은 하나의 공통 문자열 `이 서비스 화면으로 이동`을 사용하고 있었다. 이 렌더링 로직은 예약 캘린더, 댓글 목록, 상품 옵션 선택 카드에 그대로 반복 적용되므로, 실제 화면에서는 이름이 같은 버튼 3개가 생성되는 구조였다.

### 2.2 수행 탭 안의 중복 이동 제어

비교안 B 수행 탭에는 실제 서비스 구조를 설명하기 위한 이동 제어가 과도하게 겹쳐 있었다.

| 서비스 | 이전 단계의 페이지 안 빠른 이동 제어 수 | 이번 단계 후 빠른 이동 제어 수 |
|---|---:|---:|
| 예약 캘린더 | 4 | 1 |
| 댓글 목록 | 4 | 1 |
| 상품 옵션 선택 | 3 | 1 |

여기서 빠른 이동 제어 수는 서비스 콘텐츠 안에서 실제로 노출되는 바로 이동 링크·버튼만 셌다. 수행 탭 맨 아래의 보조 영역 버튼은 공통 도우미로 분리했으므로 이 표에 포함하지 않았다.

### 2.3 수행 탭 안의 비서비스 설명 문구

실제 서비스 내용과 직접 관련이 없는 설명 문구도 남아 있었다.

| 항목 | 이전 단계 | 이번 단계 후 |
|---|---:|---:|
| 예약 캘린더 설명 문구 | 1 | 0 |
| 댓글 목록 설명 문구 | 1 | 0 |
| 상품 옵션 선택 설명 문구 | 2 | 0 |

정리한 문구 예시는 다음과 같다.

- `비교안 A와 B는 같은 정보량을 유지하고, 이동 구조만 다르게 구성했습니다.`
- `댓글 내용은 같고 이동 구조만 다르게 구성했습니다.`
- `같은 상품 정보에 옵션 줄 단위의 순차 이동 구조를 적용했습니다.`
- `옵션 묶음에 한 번씩 들어간 뒤 방향키로 고르는 구조입니다.`

## 3. 적용한 변경

### 3.1 홈 화면 버튼 이름 구분 가능화

서비스 카드 버튼을 다음처럼 바꿨다.

- `예약 캘린더 화면으로 이동`
- `댓글 목록 화면으로 이동`
- `상품 옵션 선택 화면으로 이동`

이 변경은 `lib/utils.js`에 `formatServiceScreenButtonLabel()`을 추가해 공통 처리했다.

### 3.2 수행 탭 하단 보조 영역 공통화

기존에는 각 서비스 파일마다 수행 탭 하단 보조 영역을 따로 렌더링하고 있었다. 이번 단계에서는 `lib/utils.js`에 `renderRunnerFooterHtml()`을 추가해 공통화했다.

이렇게 바뀐 뒤 각 서비스 파일에는 공통 보조 영역 마크업이 남지 않고, 서비스별로 필요한 점프 버튼 이름만 전달하도록 정리했다.

### 3.3 비교안 B의 빠른 이동 제어 1개만 유지

서비스 콘텐츠 안의 빠른 이동 제어는 각 서비스별로 맨 앞의 링크 1개만 남겼다.

- 예약 캘린더: `예약 가능 시간으로 바로 이동`
- 댓글 목록: `댓글 목록으로 바로 이동`
- 상품 옵션 선택: `옵션 선택으로 바로 이동`

기존에 헤더 안이나 조건 카드 안에 중복으로 있던 이동 버튼·링크는 제거했다.

### 3.4 수행 탭 안의 설명용 문구 제거

수행 탭의 서비스 콘텐츠에서는 비교안 설명이나 실험 운영 설명을 제거했다. 실제 사용자에게 보이는 것은 서비스 내비게이션, 조건/정렬/옵션 선택, 결과 목록, 예약/댓글/장바구니 관련 UI만 남도록 정리했다.

### 3.5 수행 탭 보조 버튼 이름도 서비스별로 구체화

하단 보조 영역의 이동 버튼도 서비스별 목표를 드러내도록 바꿨다.

- 예약 캘린더: `예약 가능 시간으로 이동`
- 댓글 목록: `댓글 목록으로 이동`
- 상품 옵션 선택: `옵션 선택으로 이동`
